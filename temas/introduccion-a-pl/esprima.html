# Esprima/Espree Examples

En el Repo ULL-ESIT-GRADOII-PL/esprima-pegjs-jsconfeu-talkopen in new window encontrará el material de esta lección. Clone este repo.

The examples in this repo use a couple of JavaScript compiler frameworks: Esprimaopen in new window and Espree.

Espreeopen in new window started out as a fork of Esprimaopen in new window v1.2.2, the last stable published released of Esprima before work on ECMAScript 6 began. Espreeopen in new window is now built on top of Acornopen in new window, which has a modular architecture that allows extension of core functionality. The goal of Espreeopen in new window is to produce output that is similar to Esprima with a similar API so that it can be used in place of Esprima.

# Example: Searching for Specific Identifiers

The file idgrep.jsopen in new window is a very simple example of using Esprima to do static analysis on JavaScript code.

It provides a function idgrep that finds the appearances of identifiers matching a search string inside the input code.

Given an input like this:

➜  esprima-pegjs-jsconfeu-talk git:(master) cat hacky.js
1
// This is a hack!
const hacky = () => {
    let hack = 3;
    return 'hacky string';
}
1
2
3
4
5

if we search for hack it produces:

➜  esprima-pegjs-jsconfeu-talk git:(master) ./idgrep.js hacky.js
1:6: const hacky = () => {
2:8:     let hack = 3;
1
2
3

Observe how the appearances of hack inside the comment or the string aren't shown
